# Sutta Texts

## Overview

The goal in this directory is to create a centralised store for the "global data" that is used throughout the project, in the context of sutta / vinaya texts, translations etc.  

It will have:  

- Pali sutta and vinaya texts (raw and marked up)  
- Translated sutta and vinaya texts (raw and marked up)  
- Blurbs  
- Comments
