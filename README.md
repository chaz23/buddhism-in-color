# *buddhism in color*

A website for communicating Buddhist ideas and content, with emphasis placed on being visually appealing and being in alignment with the [Early Buddhist](https://en.wikipedia.org/wiki/Pre-sectarian_Buddhism) texts.

## Contributing

Contributions are welcomed for this project. Reach out via the SuttaCentral [Discuss and Discover](https://discourse.suttacentral.net/) site by sending me (@chaz) a personal message to see how you might be able to get involved.
